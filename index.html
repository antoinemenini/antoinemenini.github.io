    Rappels

    Ruby on Rails

    Modèles

    Générer un nouveau modèle

    $ rails generate model Product name:string description:text

    Convention de nommage :

    Au singulier et commence par une majuscule

    Types supportés :

    • :binary
    • :boolean
    • :date
    • :datetime
    • :decimal
    • :float
    • :integer
    • :primary_key
    • :string
    • :text
    • :time
    • :timestamp

    Créer Migration

    Ajouter / supprimer une colonne

    Nom de la migration de la forme "AddXXXToYYY" ou "RemoveXXXFromYYY" et suivi par une liste de noms de colonnes et de types :

    $ rails generate migration AddPartNumberToProducts part_number:string

    génère :

    class AddPartNumberToProducts < ActiveRecord::Migration
  def change
    add_column :products, :part_number, :string
  end
end

    Pour ajouter plusieurs colonnes :

    $ rails generate migration AddDetailsToProducts part_number:string price:decimal

    génère :

    class AddDetailsToProducts < ActiveRecord::Migration
  def change
    add_column :products, :part_number, :string
    add_column :products, :price, :decimal
  end
end

    $ rake db:migrate

    Contrôleurs

    Créer un nouveau contrôleur

    $ rails generate controller name method1 method2

    Convention de nommage :

    Au pluriel, commence par une majuscule. Ex : Users va créer la classe UsersController

    Console

    $ rails console

    On utilise ensuite les mêmes méthodes que dans le code pour chercher/créer/détruire des objets

    Executer une requette SQL

    > ActiveRecord::Base.connection.execute("SQL query")
